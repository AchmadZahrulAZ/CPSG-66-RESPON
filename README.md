# CPSG-66-RESPON
Halaman dashboard admin dari web RESPON

## Cara Menggunakan
* Install semua packages yang ada di devDependencies dan dependencies pada package.json

```sh
npm install
```

* Jalankan project di local server
```sh
npm run start-dev
```
